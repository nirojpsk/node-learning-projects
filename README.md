# Doctor Appointment System (Backend)

A Node.js + Express + MongoDB backend project.

## Features
- User authentication (JWT)
- Doctor approval system
- Appointment booking
- Role-based access (Admin / Doctor / Patient)

## Tech Stack
- Node.js
- Express
- MongoDB
- JWT
- Mongoose

## How to Run
1. npm install
2. create `.env` using `.env.example`
3. npm run server
